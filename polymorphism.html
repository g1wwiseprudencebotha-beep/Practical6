Polymorphism

Polymorphism meaning many forms.It enables objects of different classes to be treated as instances of a common superclass,allowing a single intrfaces to represent different underlying forms (datatype)

Example

Shape base on class has a draw() method,but subclasses Circle,Square, and Triangle each implement draw() differently.When calling draw() on list of shapes, each object acts according to its specific type. Benefits
  • Enhances code flexibility,
  • Reusability
    • Maintainability
