Why Use Inheritance?

  • Code Reusability
  • Reduces duplication
  • Makes programs easier to maintain
  • Creates logical relationship between classes

Real-Life Example

Think of a vehicleas a parent class. A CarandMotorcycleare child classes. They inherit common features like speed and fuel,but they also have their own special features.

Java Example

class Vehicle{
int speed;

void move(){
System.out.println("The vehicle is moving");
 }
}

class Car extend Vehicle{
int wheels=4;

void display(){
System.out.println("Car has"+ wheels +"wheels");
} 
}

Here, the Carclass inherits the speed variables and move()method from Vehicleclass.

Types of Inheritance

  • Single Inheritance
  • Multilevel Inheritance
  • Hierachical Inheritance

Conclusion

Inheritance allows classes to build relationships with each other. It promotes code reuse and helps developers create structured, organized programs.
