What is Class?

A class is a blueprint or template used to create objects. It defines the properties (variables) and behaviors (methods) that the objects created form it will have.

Example in Java:

class Car{
String color;
int speed;

void drive(){
System.out.printin("Driving at" +speed+"km/h");
}
}

In this example:
.Caris the class
.color and speed are atrributes
.drive() is a method
